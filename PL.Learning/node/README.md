## [Node.JS](#)
**介绍**：Node.js 是一个开源的、跨平台的、基于Chrome V8引擎、C++绑定层、C语言异步IO库 `libuv` 的可以和操作系统交互的JavaScript**运行时环境**，其特点是: **事件驱动**、**单线程**、**非阻塞异步 I/O**。
